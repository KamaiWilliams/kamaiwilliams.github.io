Chop&Screw

2026

An interactive loop sequencer inspired by the chopped up, high energy sound of jerk music. Built with HTML, CSS, and JavaScript, the project allows users to place, move, and remove sound “dots” on a looping grid while experimenting with BPM and timing through a snap-to-beat system. The interface explores how rhythm, motion, and interaction design can create an intuitive and playful music-making experience.

Reach 4 Stars

2025

An interactive camera-based experience that transforms hand therapy exercises into playful gameplay. Using real-time hand tracking, users interact with responsive star-like forms through gestures such as finger wiggling, curling, and squeezing. The project explores how movement, feedback, and visual reward can make rehabilitative exercises for carpal tunnel relief more engaging and intuitive. ​​​​​​​
