Data Visualization

Uncontained

2025

An interactive data visualization exploring public restroom accessibility across New York City. Users can filter restrooms by borough, explore mapped locations, and better understand spatial disparities in public amenities. Built from scratch using HTML, CSS, JavaScript, and D3.js, the project focuses on visual clarity, accessibility, and user interaction.

Housing Rights Initiative Non-Profit

2026

An interactive storytelling and data visualization project exploring source-of-income discrimination against Section 8 voucher holders. Working within a multidisciplinary team, we transformed fair housing audit data into a web experience combining real text conversations, mapping, and layered narrative pacing. I focused on UX design, site structure, information hierarchy, and translating dense research into an accessible visual system.
