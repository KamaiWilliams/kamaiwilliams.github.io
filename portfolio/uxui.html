Chop&Screw

Interactive Loop Sequencer Built with HTML, CSS, and JavaScript in VS Code Chop & Screw is an interactive loop-making tool inspired by the chopped-up, high-energy sound of jerk music. Users can drop, move, and delete sound “dots” on a looping grid, change the BPM, and experiment with timing using a “Snap to Beat” feature that mimics how real sequencers work. This project let me blend my love for music and design while figuring out how to make timing, rhythm, and motion feel fun and intuitive through code. https://kamaiwilliams.github.io/chopandscrew/index.html

Reach 4 Stars

Carpal Tunnel relief game An interactive, camera-based experience that turns physical hand therapy exercises into playful, responsive gameplay. Using real-time hand tracking, users interact with star-like forms through different gestures like finger wiggling, curling, and squeezing. The project explores how movement, feedback, and visual reward can transform Carpal Tunnel relief rehabilitative exercises into an engaging, intuitive interface. https://kamaiwilliams.github.io/hand-game/​​​​​​​
