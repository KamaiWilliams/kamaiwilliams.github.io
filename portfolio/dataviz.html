Public Restroom Accessibility Map This interactive data visualization explores the accessibility of public restrooms across New York City. Users can filter restrooms by borough, see their locations plotted on a custom map, and interact with the interface to better understand spatial disparities in public amenities. The project was fully coded from scratch using HTML, CSS, and JavaScript, and it marked my first experience working with D3.js for data-driven mapping and visualization. I built and tested the entire project in Visual Studio Code, focusing on both visual clarity and user interaction. https://kamaiwilliams.github.io/uncontained/index.html

Housing Discrimination Initiative Non-Profit

2026

This project explored source-of-income discrimination against Section 8 voucher holders through interactive storytelling, mapping, and data visualization. Working with a multidisciplinary team, we transformed fair housing audit data into a web experience that combined real text conversations, mapped case outcomes, and layered narrative pacing. I led the planning of the site structure and user flow, developing the framework for how users would navigate and understand the data. My role centered on UX design, information hierarchy, and translating dense research into an accessible visual system. Web link: https://ece-476-interactive-article.vercel.app
